##Aplikasi To-Do List

##Deskripsi
Aplikasi sederhana berbasis PHP untuk mencatat tugas harian.

##Fitur
-Tambah Tugas
-Hapus Tugas
-Edit Tugas
-Tandai Tugas Selesai

##Struktur Folder
-index.php - halaman utama

##Cara Menjalankan
1. salin folder ke 'htdocs/'
2. jalankan XAMPP dan buka di 'http://localhost/index.php'
